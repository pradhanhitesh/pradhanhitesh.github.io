PROJECTS

A collection of projects I've worked on.
FENS

I used Python, HTML, and GitHub Actions to automatically fetch new job postings on Federation of Neuroscience (FENS) job market every week and email the results to specified recipients as PDF file.

3D-MRI-Transformations

I am currently reviewing literally to find state-of-the-art 3D transformation techniques to build better and efficient 2D CNN models, instead of relying on data-hungry 3D CNN which are difficult to train.

Characteristics Table

I used R and ShinyApps to deploy the an user-friendly web interface to generate publication-ready characteristics tables that allows user to visualize the differences between two study groups.

Neuroimaging Plugins

As I am working with the neuroimaging softwares, I have created small plugins for some popular softwares to help ease out some functionality which I think makes the workflow easy.
